Welcome to Fakemail's documentation!
====================================

**Fakemail** is a Python library for generating fake email addresses.
It helps you create random email addresses for testing and development purposes.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is for testing purposes.
